# しんめ.rb

## 概要

しんめ.rbは異職種からエンジニアになって半年の@maimux2xがプログラミング実務経験2年未満くらいの人たちでLTとかもくもく会じゃなくて実際に手を動かして勉強したり困りごとを共有するコミュニティがあったら嬉しいなと思い立ち上げました。
名前の由来は「新芽」で、コミュニティとともに大きく成長していきたいという意味が込められています。
